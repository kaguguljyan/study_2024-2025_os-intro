# Цель работы

Изучить идеологию и применение средств контроля версий. Освоить умения
по работе с git.

# Задание

1.  Установка git.
2.  Установка gh.
3.  Базовая настройка git.
4.  Создайте ключи ssh.
5.  Настройка gh.
6.  Создание репозитория курса на основе шаблона.
7.  Настройка каталога курса.
